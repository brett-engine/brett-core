# AGENTS

- brett-core is Rust library to represent General Game Playing (GGP) boards and games
- `mise` (`mise.toml`) for dev-dependencies and as a task runner (should be installed)
- `cargo` (`Cargo.toml`) for Rust dependencies

## Development

- Run `mise install` before starting work to ensure dev-dependencies are installed
- Run `mise run fix` after finishing work to apply formatting and auto-fixable suggestions from checkers
- Run `mise run check` after `mise run fix` to list issues that cannot be fixed automatically
